# Métodos Quantitativos – Análise de passagens aéreas

Este repositório reúne os arquivos utilizados no trabalho da disciplina Métodos Quantitativos Aplicados à Administração.

O estudo analisa a variação dos preços de passagens aéreas de ida no trecho Salvador (SSA) – Natal (NAT), considerando variáveis como preço da passagem, antecedência da compra, duração total do voo, número de escalas e quantidade de voos disponíveis.

## Estrutura do repositório

- `amostra.csv`: base de dados da amostra utilizada nas análises estatísticas.
- `analises.ipynb`: notebook com os cálculos estatísticos realizados no trabalho, incluindo Alfa de Cronbach, teste de aleatoriedade, estatísticas descritivas, agrupamento K-means, correlação de Pearson e regressão linear múltipla.
- `scraper_ssa_nat_8datas_150dias_id_voo.py`: script utilizado como apoio à coleta automatizada dos dados no Google Flights para o trecho Salvador (SSA) – Natal (NAT).
- `boxplots/`: pasta contendo os gráficos boxplot gerados para as variáveis quantitativas da amostra.

## Técnicas utilizadas

- Estatística descritiva
- Alfa de Cronbach
- Teste de Runs
- Agrupamento K-means
- Correlação de Pearson
- Regressão linear múltipla

## Observação

Os dados foram coletados para fins exclusivamente acadêmicos, em uma única data de busca, conforme descrito na metodologia do trabalho.
